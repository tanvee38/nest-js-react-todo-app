# TodoList api

```cd api```
```npm install```
```npm run build```
```docker compose up --build```


Go to `http://localhost:3000/todos` to see the existing todos.


## Todo list web app

```cd todo-web-app```
```npm install```
to build web app: ```npm run build```
To run web app: ```npm run server```

Go to `http://localhost:8082` to run the project on the browser.
